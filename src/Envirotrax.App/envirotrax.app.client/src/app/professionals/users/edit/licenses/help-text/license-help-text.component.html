Please read the following important information on how to register and manage your licenses

Adding Licenses

Use the form below to add a new license for this professional. All license types, including CSI Inspector, BPAT, FOG Inspector, and others, are managed from this single location.

Each license must be registered for the inspector it belongs to. The name on the inspector's account must match the name on the license.

Once submitted, all licenses will be verified by Vepo personnel. Our staff will validate the license by entering the expiration date on file with the authority that issued the license.

Renewing Your Licenses

All licenses will be monitored by Vepo personnel and updated near the time of their expiration. If the license is not renewed, we will not be able to update it and the inspector's account will become unauthorized for use.

Reinstating Expired Licenses

Vepo personnel will continue to check for renewals of expired licenses for a set period of time. If the license is not renewed within that period, it will no longer be checked for renewal and must be reinstated manually. To reinstate an expired license, first renew it with the issuing authority, then contact us so we can verify and update it in the system.
